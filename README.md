# CO2-Emissionen-Webseite IPWA01-01 Aufgabe 1
EmissionenImBlick.de
Projektübersicht

Dieses Projekt ist eine im Rahmen einer Fallstudie entwickelte Webseite zur Darstellung fiktiver CO2-Emissionsdaten von Ländern und Unternehmen. Ziel ist es, Grundlagen moderner Webentwicklung sowie responsive Layouts und einfache Interaktivität umzusetzen.

Funktionen

- Responsives Webdesign (Desktop, Tablet, Smartphone)
- Globale und lokale Navigation
- Interaktive Tabelle mit Filterfunktion nach Land und Unternehmen
- Einfache Sortierung der CO₂-Emissionswerte

Verwendete Technologien

- HTML – Struktur der Webseite
- CSS / Tailwind CSS – Layout und Design
- JavaScript – Filter- und Tabellenlogik
- Visual Studio Code – Entwicklungsumgebung
